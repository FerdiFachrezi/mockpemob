  # Lapo Kerja

Lapo Kerja adalah aplikasi Android berbasis marketplace jasa yang mempertemukan Klien (pencari jasa) dengan Pekerja (penyedia jasa) secara mudah dan cepat. Aplikasi ini masih dalam tahap Alpha, sehingga masukan dan saran sangat diharapkan untuk pengembangan selanjutnya.

---

## Status Aplikasi

Versi: Alpha

Catatan:

* Beberapa fitur masih dalam tahap pengembangan
* Kemungkinan terdapat bug atau keterbatasan performa

---

## Fitur Utama

* Registrasi dan login menggunakan Email & Password
* Login cepat menggunakan akun Google
* Pemilihan peran sebagai Klien atau Pekerja
* Pencarian pekerja berdasarkan kategori dan kata kunci
* Sistem pemesanan jasa (booking)
* Integrasi WhatsApp untuk komunikasi
* Sistem ulasan dan rating

---

## Instalasi

1. Unduh file APK dari repository GitHub ini
2. Buka file APK di perangkat Android
3. Jika muncul peringatan keamanan, aktifkan opsi Install from Unknown Sources
4. Lanjutkan proses instalasi hingga selesai
5. Buka aplikasi Lapo Kerja

---

## Panduan Penggunaan

### Langkah Awal

1. Buka aplikasi dan tunggu Splash Screen
2. Masuk ke halaman Selamat Datang
3. Pilih peran:

   * Saya Klien untuk mencari jasa
   * Saya Pekerja untuk menawarkan jasa
4. Daftar akun menggunakan Email & Password atau login dengan Google

---

## Panduan Klien (Pencari Jasa)

Tujuan: mencari pekerja, memesan jasa, dan memberi ulasan

### Mencari Pekerja

* Pilih kategori jasa di Beranda
* Gunakan kolom pencarian untuk layanan spesifik
* Lihat rekomendasi pada bagian Layanan Populer atau Pekerja Favorit

### Memesan Jasa

1. Pilih pekerja untuk melihat profil lengkap
2. Klik Pesan Jasa
3. Isi detail pemesanan:

   * Tanggal
   * Jam
   * Alamat lengkap
   * Catatan tambahan
4. Klik Buat Pesanan

### Menunggu Konfirmasi

* Buka menu Pesanan
* Cek tab Baru dengan status Menunggu
* Status berubah menjadi Diproses setelah pekerja menerima pesanan

### Komunikasi

* Buka detail pesanan
* Gunakan tombol Telepon atau Chat untuk terhubung via WhatsApp

### Menyelesaikan Pesanan dan Ulasan

* Setelah pekerjaan selesai, status menjadi Selesai
* Klik Beri Ulasan
* Berikan rating bintang dan komentar

---

## Panduan Pekerja (Penyedia Jasa)

Tujuan: menerima order, menyelesaikan pekerjaan, dan membangun reputasi

### Melengkapi Profil

Wajib mengisi:

* Kategori layanan
* Tarif (standar atau nego)
* Keahlian
* Deskripsi diri

Profil ini akan ditampilkan kepada klien

### Menerima Orderan

1. Buka menu Pekerjaan
2. Cek tab Baru
3. Lihat detail pesanan
4. Pilih Terima Pesanan atau Tolak

### Proses Pekerjaan

* Pesanan yang diterima masuk ke tab Aktif
* Datang sesuai jadwal dan lokasi
* Gunakan WhatsApp untuk komunikasi jika diperlukan

### Menyelesaikan Pekerjaan

* Buka detail pesanan
* Klik Tandai Selesai
* Status pesanan berubah menjadi selesai

### Melihat Reputasi

* Buka menu Profil
* Pilih Lihat Ulasan Saya
* Rating dan komentar dari klien akan ditampilkan

---

## Fitur Tambahan
